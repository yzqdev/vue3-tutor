### API
| 属性      | 说明                                    | 类型           | 默认值 |
|-----------|-----------------------------------------|----------------|--------|
| count     | 显示的文字                              | String，Number | -      |
| color     | 徽标颜色                                | String         | -      |
| max-count | 展示封顶的数字值，高于的部分会以+号显示 | Number         | 99     |
| dot       | 不展示数字，只有一个小红点              | Boolean        | false  |
