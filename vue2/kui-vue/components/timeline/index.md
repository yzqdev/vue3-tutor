### API
| 属性  | 说明                                                                                   | 类型        | 默认值 |
| ----- | -------------------------------------------------------------------------------------- | ----------- | ------ |
| icon  | 时间轴 `item` 图标                                                                     | String      | -      |
| color | 时间轴 `item` 图标颜色                                                                 | String      | -      |
| time  | 时间文本                                                                               | String      | -      |
| extra | 自定义辅助内容                                                                         | String,slot | -      |
| mode  | 通过设置 `mode` 可以改变时间轴和内容的相对位置	`left` ,`center`, `alternate` , `right` | String      | left   |
