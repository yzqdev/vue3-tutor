#### API
| 属性        | 说明                                         | 类型            | 默认值 |
|-------------|----------------------------------------------|-----------------|--------|
| description | 自定义描述内容                               | [String、slot ] | -      |
| imageStyle  | 图片样式                                     | Object          | -      |
| image       | 设置显示图片，为 string 时表示自定义图片地址 | [String、slot ] | -      |
